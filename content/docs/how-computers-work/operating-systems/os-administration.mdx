### 3.2 File System Permissions

<Tabs items={["ACL Basics", "Permission Types", "Inheritance", "Ownership"]}>
  <Tab>
    <div className="p-4 border rounded-lg bg-white dark:bg-gray-800 my-4">
      <h4 className="font-bold mb-2 text-blue-600 dark:text-blue-400">Access Control Lists (ACLs)</h4>
      
      <p>ACLs define who can access resources and what they can do with them. The process follows these steps:</p>
      
      <Steps>
        <Step>
          <h5>Access Request</h5>
          <p>User requests access to a resource</p>
        </Step>
        <Step>
          <h5>SID Check</h5>
          <p>System checks user's Security Identifier in ACL</p>
        </Step>
        <Step>
          <h5>Group Check</h5>
          <p>If no user SID, system checks group memberships</p>
        </Step>
        <Step>
          <h5>Permission Evaluation</h5>
          <p>System evaluates allowed actions for matching SIDs</p>
        </Step>
        <Step>
          <h5>Access Decision</h5>
          <p>System grants or denies access based on ACL entries</p>
        </Step>
      </Steps>
    </div>
  </Tab>

  <Tab>
    <div className="p-4 border rounded-lg bg-white dark:bg-gray-800 my-4">
      <h4 className="font-bold mb-2 text-green-600 dark:text-green-400">Permission Types</h4>
      
      <div className="grid grid-cols-1 md:grid-cols-2 gap-4">
        <div className="p-4 border rounded-lg bg-gray-50 dark:bg-gray-900">
          <h5 className="font-bold">Explicit Permissions</h5>
          <ul className="list-disc pl-5">
            <li>Directly assigned to resource</li>
            <li>Override inherited permissions</li>
            <li>Highest priority in conflicts</li>
            <li>Manually configured</li>
          </ul>
        </div>
        
        <div className="p-4 border rounded-lg bg-gray-50 dark:bg-gray-900">
          <h5 className="font-bold">Inherited Permissions</h5>
          <ul className="list-disc pl-5">
            <li>Received from parent object</li>
            <li>Lower priority than explicit</li>
            <li>Automatically propagated</li>
            <li>Shown in gray in Windows</li>
          </ul>
        </div>
      </div>
      
      <div className="mt-4">
        <h5 className="font-bold">Permission Priority (Highest to Lowest)</h5>
        <ol className="list-decimal pl-5">
          <li>Explicit Deny</li>
          <li>Explicit Allow</li>
          <li>Inherited Deny</li>
          <li>Inherited Allow</li>
        </ol>
      </div>
    </div>
  </Tab>

  <Tab>
    <div className="p-4 border rounded-lg bg-white dark:bg-gray-800 my-4">
      <h4 className="font-bold mb-2 text-purple-600 dark:text-purple-400">Permission Inheritance</h4>
      
      <Accordions>
        <Accordion title="Inheritance Basics">
          <ul className="list-disc pl-5">
            <li>New objects inherit parent permissions</li>
            <li>Ensures consistent access control</li>
            <li>Simplifies permission management</li>
            <li>Applies to subfolders and files</li>
          </ul>
        </Accordion>
        
        <Accordion title="Breaking Inheritance">
          <p>Two options when breaking inheritance:</p>
          <ul className="list-disc pl-5">
            <li>Copy permissions (convert to explicit)</li>
            <li>Remove inherited permissions</li>
          </ul>
          <p className="mt-2 text-yellow-600 dark:text-yellow-400">Warning: Maintain system SIDs (SYSTEM, CREATOR OWNER) for proper operation</p>
        </Accordion>
        
        <Accordion title="Replacing Child Permissions">
          <p>The "Replace all child object permissions" option:</p>
          <ul className="list-disc pl-5">
            <li>Forces inheritance on all children</li>
            <li>Overwrites existing permissions</li>
            <li>Affects all subfolders and files</li>
            <li>Cannot be easily undone</li>
          </ul>
        </Accordion>
      </Accordions>
    </div>
  </Tab>

  <Tab>
    <div className="p-4 border rounded-lg bg-white dark:bg-gray-800 my-4">
      <h4 className="font-bold mb-2 text-red-600 dark:text-red-400">Resource Ownership</h4>
      
      <div className="grid grid-cols-1 md:grid-cols-2 gap-4">
        <div className="p-4 border rounded-lg bg-gray-50 dark:bg-gray-900">
          <h5 className="font-bold">Owner Rights</h5>
          <ul className="list-disc pl-5">
            <li>Full control over resource</li>
            <li>Can modify permissions</li>
            <li>Can transfer ownership</li>
            <li>Cannot be locked out</li>
          </ul>
        </div>
        
        <div className="p-4 border rounded-lg bg-gray-50 dark:bg-gray-900">
          <h5 className="font-bold">CREATOR OWNER</h5>
          <ul className="list-disc pl-5">
            <li>Special security principal</li>
            <li>Represents resource creator</li>
            <li>Automatic assignment</li>
            <li>Inheritable permissions</li>
          </ul>
        </div>
      </div>
    </div>
  </Tab>
</Tabs>
