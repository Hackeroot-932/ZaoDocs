## Programming Paradigms

<Tabs items={["Procedural", "Object-Oriented", "Functional", "Event-Driven"]}>
  <Tab>
    <div className="p-4 border rounded-lg bg-white dark:bg-gray-800 my-4">
      <h4 className="font-bold mb-2 text-blue-600 dark:text-blue-400">Procedural Programming</h4>
      <p>A programming paradigm based on the concept of procedure calls, where programs are structured as sequences of instructions grouped into procedures.</p>
      
      <ul className="list-disc pl-5 mt-2">
        <li>Emphasizes breaking down programs into procedures (functions)</li>
        <li>Uses a top-down approach to problem solving</li>
        <li>Focuses on data and operations separately</li>
        <li>Examples: C, Pascal, BASIC</li>
      </ul>
    </div>
  </Tab>
  <Tab>
    <div className="p-4 border rounded-lg bg-white dark:bg-gray-800 my-4">
      <h4 className="font-bold mb-2 text-green-600 dark:text-green-400">Object-Oriented Programming</h4>
      <p>A programming paradigm based on the concept of "objects" that contain data and code.</p>
      
      <ul className="list-disc pl-5 mt-2">
        <li>Organizes code into objects that contain both data and behavior</li>
        <li>Supports inheritance, encapsulation, and polymorphism</li>
        <li>Promotes code reusability and modularity</li>
        <li>Examples: Java, C++, Python</li>
      </ul>
    </div>
  </Tab>
  <Tab>
    <div className="p-4 border rounded-lg bg-white dark:bg-gray-800 my-4">
      <h4 className="font-bold mb-2 text-purple-600 dark:text-purple-400">Functional Programming</h4>
      <p>A programming paradigm that treats computation as the evaluation of mathematical functions.</p>
      
      <ul className="list-disc pl-5 mt-2">
        <li>Emphasizes immutable data and pure functions</li>
        <li>Avoids changing state and mutable data</li>
        <li>Supports higher-order functions and recursion</li>
        <li>Examples: Haskell, Lisp, Erlang</li>
      </ul>
    </div>
  </Tab>
  <Tab>
    <div className="p-4 border rounded-lg bg-white dark:bg-gray-800 my-4">
      <h4 className="font-bold mb-2 text-red-600 dark:text-red-400">Event-Driven Programming</h4>
      <p>A programming paradigm where the flow of the program is determined by events.</p>
      
      <ul className="list-disc pl-5 mt-2">
        <li>Program flow is determined by user actions or system events</li>
        <li>Widely used in graphical user interfaces</li>
        <li>Supports asynchronous programming</li>
        <li>Examples: JavaScript, Visual Basic</li>
      </ul>
    </div>
  </Tab>
</Tabs>
