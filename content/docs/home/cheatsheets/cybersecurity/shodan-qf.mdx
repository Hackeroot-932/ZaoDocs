# Shodan Query Filters Cheatsheet

<Callout title="What is Shodan?" type="info">
  Shodan is a search engine for Internet-connected devices (IoT, servers, industrial equipment, etc.). Unlike traditional search engines that index web content, Shodan scans the Internet for devices, collecting information about their configuration, services, and potential vulnerabilities.
</Callout>

## 4. CLI Commands

<Tabs items={["Basic Commands", "Search Commands", "Analysis Commands", "Download Commands"]}>
  <Tab>
    <div className="p-4 border rounded-lg bg-white dark:bg-gray-800 my-4">
      <h4 className="font-bold mb-2 text-blue-600 dark:text-blue-400">Basic CLI Commands</h4>
      
      ```bash
      # Show host information
      shodan host 8.8.8.8

      # Show your API info
      shodan info

      # Count results
      shodan count apache
      ```
    </div>
  </Tab>
  <Tab>
    <div className="p-4 border rounded-lg bg-white dark:bg-gray-800 my-4">
      <h4 className="font-bold mb-2 text-green-600 dark:text-green-400">Search Commands</h4>
      
      ```bash
      # Basic search
      shodan search apache

      # Search with filters
      shodan search "product:nginx country:DE"

      # Search with specific fields
      shodan search --fields ip_str,port,org apache
      ```
    </div>
  </Tab>
  <Tab>
    <div className="p-4 border rounded-lg bg-white dark:bg-gray-800 my-4">
      <h4 className="font-bold mb-2 text-purple-600 dark:text-purple-400">Analysis Commands</h4>
      
      ```bash
      # Get statistics
      shodan stats --facets country apache

      # Check if IP is a honeypot
      shodan honeyscore 1.2.3.4

      # Scan your network
      shodan scan submit 1.2.3.0/24
      ```
    </div>
  </Tab>
  <Tab>
    <div className="p-4 border rounded-lg bg-white dark:bg-gray-800 my-4">
      <h4 className="font-bold mb-2 text-red-600 dark:text-red-400">Download Commands</h4>
      
      ```bash
      # Download search results
      shodan download results.json.gz apache

      # Parse downloaded data
      shodan parse results.json.gz

      # Convert data to different format
      shodan convert results.json.gz csv
      ```
    </div>
  </Tab>
</Tabs>
